Informativa sulla Privacy

RicordAI

Ultimo aggiornamento: Giugno 2026

🔒 RicordAI non raccoglie, non archivia e non trasmette i tuoi dati personali a server propri. Tutti i dati rimangono sul tuo dispositivo.

1. Titolare del trattamento

Il titolare del trattamento dei dati è il team di sviluppo di RicordAI, contattabile all'indirizzo email: privacy@ricordai.app

2. Dati raccolti e come vengono utilizzati

RicordAI raccoglie e tratta le seguenti categorie di dati, tutti salvati esclusivamente sul tuo dispositivo:

  • Nome utente — inserito volontariamente durante l'onboarding, usato per personalizzare l'esperienza
  • Scadenze ed eventi — creati dall'utente, salvati localmente tramite AsyncStorage
  • Foto di documenti — scattate o caricate dall'utente per la funzione Carte & Documenti, salvate esclusivamente sul dispositivo (solo per utenti Premium)
  • Preferenze — lingua, tema, sfondo, gruppi attivi
  • Storico chat AI — le conversazioni con l'assistente AI, salvate localmente

3. Dati che NON raccogliamo

RicordAI non raccoglie:

  • Indirizzi IP o identificatori di rete
  • Dati di geolocalizzazione
  • Dati di navigazione o profilazione
  • Informazioni finanziarie o di pagamento
  • Contatti della rubrica
  • Qualsiasi dato su server propri

4. Servizi di terze parti

RicordAI si avvale di servizi di terze parti per alcune funzionalità:

ANTHROPIC API

Per generare risposte dell'assistente AI, il testo delle domande viene inviato alle API di Anthropic (Claude). Anthropic non utilizza i dati delle API per addestrare i propri modelli. I log tecnici vengono conservati temporaneamente per scopi di sicurezza. Per maggiori informazioni: anthropic.com/privacy

COMPARATORI ASSICURATIVI

Quando l'utente clicca su un link di comparazione (Segugio.it, Facile.it, MutuiOnline ecc.), viene reindirizzato al sito esterno. Da quel momento si applicano le privacy policy dei rispettivi siti. RicordAI potrebbe ricevere una commissione di affiliazione per i contratti stipulati.

APPLE (APNs)

Le notifiche push vengono inviate tramite Apple Push Notification Service (APNs). Apple gestisce la consegna delle notifiche secondo la propria privacy policy.

5. Sicurezza dei dati

I dati dell'utente sono protetti da:

  • Sandbox iOS — i dati sono inaccessibili ad altre app
  • Autenticazione biometrica — Face ID / Touch ID all'apertura dell'app
  • Archiviazione locale — nessun dato transita su server propri di RicordAI

6. Diritti dell'utente (GDPR)

In conformità al Regolamento Europeo 2016/679 (GDPR), l'utente ha diritto a:

  • Accesso — visualizzare i propri dati (tutti visibili direttamente nell'app)
  • Rettifica — modificare i propri dati in qualsiasi momento dall'app
  • Cancellazione — eliminare tutti i dati tramite Impostazioni → Elimina tutti i dati
  • Portabilità — i dati sono salvati localmente e accessibili tramite backup del dispositivo

Per qualsiasi richiesta relativa ai dati personali: privacy@ricordai.app

7. Conservazione dei dati

I dati vengono conservati sul dispositivo dell'utente fino a quando l'utente non li elimina manualmente o disinstalla l'app. RicordAI non ha accesso ai dati dell'utente in nessun momento.

8. Minori

RicordAI non è destinata a utenti di età inferiore a 13 anni. Non raccogliamo consapevolmente dati di minori.

9. Modifiche alla Privacy Policy

Eventuali modifiche alla presente Privacy Policy saranno pubblicate su questa pagina con data di aggiornamento. L'uso continuato dell'app dopo le modifiche costituisce accettazione della nuova policy.

10. Contatti

Per qualsiasi domanda relativa alla privacy: privacy@ricordai.app

🔒 RicordAI does not collect, store or transmit your personal data to any servers. All data remains on your device.

1. Data Controller

The data controller is the RicordAI development team, reachable at: privacy@ricordai.app

2. Data Collected and How It Is Used

RicordAI collects and processes the following categories of data, all stored exclusively on your device:

  • Username — voluntarily entered during onboarding, used to personalize the experience
  • Deadlines and events — created by the user, stored locally via AsyncStorage
  • Document photos — taken or uploaded by the user for the Cards & Documents feature, stored exclusively on the device (Premium users only)
  • Preferences — language, theme, background, active groups
  • AI chat history — conversations with the AI assistant, stored locally

3. Data We Do NOT Collect

RicordAI does not collect:

  • IP addresses or network identifiers
  • Geolocation data
  • Browsing or profiling data
  • Financial or payment information
  • Address book contacts
  • Any data on our own servers

4. Third-Party Services

RicordAI uses third-party services for certain features:

ANTHROPIC API

To generate AI assistant responses, question text is sent to Anthropic's APIs (Claude). Anthropic does not use API data to train its models. Technical logs are temporarily retained for security purposes. For more information: anthropic.com/privacy

INSURANCE COMPARATORS

When a user clicks a comparison link (Segugio.it, Facile.it, MutuiOnline etc.), they are redirected to the external site. The respective site's privacy policies apply from that point. RicordAI may receive an affiliate commission for contracts signed.

APPLE (APNs)

Push notifications are sent via Apple Push Notification Service (APNs). Apple handles notification delivery according to its own privacy policy.

5. Data Security

User data is protected by:

  • iOS Sandbox — data is inaccessible to other apps
  • Biometric authentication — Face ID / Touch ID when opening the app
  • Local storage — no data passes through RicordAI's own servers

6. User Rights (GDPR)

In compliance with European Regulation 2016/679 (GDPR), users have the right to:

  • Access — view their data (all visible directly in the app)
  • Rectification — modify their data at any time from the app
  • Erasure — delete all data via Settings → Delete all data
  • Portability — data is stored locally and accessible via device backup

For any data-related requests: privacy@ricordai.app

7. Data Retention

Data is retained on the user's device until the user manually deletes it or uninstalls the app. RicordAI has no access to user data at any time.

8. Children

RicordAI is not intended for users under the age of 13. We do not knowingly collect data from minors.

9. Changes to This Privacy Policy

Any changes to this Privacy Policy will be published on this page with an updated date. Continued use of the app after changes constitutes acceptance of the new policy.

10. Contact

For any privacy-related questions: privacy@ricordai.app
